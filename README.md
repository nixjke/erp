## Инструкция по запуску

### Сборка

```bash
> git clone http://gitlab.wconsults.ru/ERP/mysmartoffice_app_react.git
> cd mysmartoffice_app_react
> git checkout dev
> npm install
> npm run build
```

### Разработка

```bash
> git clone http://gitlab.wconsults.ru/ERP/mysmartoffice_app_react.git
> cd mysmartoffice_app_react
> git checkout dev
> npm install
> npm start
```
